About

Bucketlist tracker is a Django powered bucketlist app for managing bucketlists, a project by Oluwafemi Sule, a software developer at Andela. He is of the opinion that you should have a list of things you hope to have accomplished before you kick the bucket.

However, he submits that this should be very easy for age-old persons to use and would appreciate any feedback that would aid this purpose here.

Bucketlist tracker comes with a web frontend and API access with great documentation which you can find here and here respectively.
